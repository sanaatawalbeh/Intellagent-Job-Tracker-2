# Intelligent Job Tracker

## App Name

-**Job Tracker**

## Features

-**Authentication System** (Login / Signup with Firebase).  
-**Admin Panel** to view and manage job applications.  
-**AI-Powered Tools**: -**Resume Feedback** → Upload your resume and get instant AI feedback to improve it.  
-**Job Analyzer** → Paste a job description and let AI highlight required skills and match them with your resume.  
-**Dashboard Insights** → AI-powered insights about your job applications and success rate.  
-**Career Chatbot** → Ask job or career questions and get AI-powered responses.  
-**Modern UI** built with React + Material UI.

## How it works

1. Users can **sign up / log in** using Firebase Authentication.
2. The **Admin Panel** provides an overview of all job applications.
3. AI services help **users**:
   -Improve their resumes.  
   -Analyze job postings.  
   -Understand application trends through AI insights.  
   -Chat with a career assistant chatbot.
4. Data persistence is handled via Firebase (auth) and a custom backend for chatbot + AI requests.
